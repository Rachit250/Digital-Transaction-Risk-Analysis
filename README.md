📁 Project Structure
```
Digital-Transaction-Risk-Analysis/
│
├── Unit1_Data_Cleaning.ipynb          # Data preprocessing & missing value treatment
├── Unit2_Descriptive_Analysis.ipynb   # EDA, distributions & visualizations
├── Unit3_Inferential_Testing.ipynb    # Hypothesis testing & statistical inference
├── Unit4_Predictive_Modeling.ipynb    # ML models for risk prediction
├── Unit5_Dimensionality_Reduction.ipynb # PCA, t-SNE & feature selection
```
